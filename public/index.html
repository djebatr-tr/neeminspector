Ağ Trafiği (En Güvenilir)

Headless Chrome sayfayı yüklerken tüm ağ isteklerini dinler. Oynatıcının yaptığı .m3u8, .mp4 gibi istekler anında yakalanır — bu yöntem token'lı ve gizli linkleri de bulur.

Embed Oynatıcılar

Vidmoly, Mixdrop, Filemoon, Streamtape gibi harici oynatıcı platformları iframe ile gömülür. Bu iframe'in src URL'i, oynatıcının gerçek adresini gösterir.

Script Tarama

JWPlayer, VideoJS gibi oynatıcıların config nesneleri JavaScript kodunda saklı olabilir. Tüm script blokları regex ile taranarak .m3u8, .mp4, .mpd linkleri çıkarılır.

Cloudflare Hakkında

Cloudflare bot koruması olan sitelerde ilk yükleme CAPTCHA gösterebilir. Puppeteer stealth modunda çalışır fakat bazı sitelerin güçlü bot korumasını aşmak mümkün olmayabilir.
